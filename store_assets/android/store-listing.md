Play Store-grænser: **titel ≤ 30 tegn**, **kort beskrivelse ≤ 80 tegn**, **fuld beskrivelse ≤ 4000 tegn**.

## 🇬🇧 English (en-US) — default

### Title (29)
```
HA Widgets for Home Assistant
```

### Short description (73)
```
Home Assistant on your home screen — fast, offline-ready native widgets.
```

### Full description
```
HA Widgets brings your Home Assistant entities straight to your Android home screen. No need to open an app — control lights, switches, scenes and more with a single tap, and read live sensor values at a glance.

The app connects directly to your own Home Assistant instance using a long-lived access token. There is no cloud service in between: your data travels only between your phone and your server.

WIDGET TYPES
• Light — toggle, or dim with a brightness slider
• Switch — simple on/off
• Scene & Script — activate with one tap
• Automation — toggle or trigger
• Sensor & Binary sensor — live values with automatic units and icons
• Climate — temperature and mode, adjustable
• Cover — open, close and set position
• Dashboard shortcut — open any Lovelace dashboard in a built-in view
• Multi-entity — several entities in one widget, with rows and secondary chips

KEY FEATURES
• Works offline — widgets show the last known state from a local cache, with a staleness marker
• Battery-friendly — native background sync instead of a constant connection
• Secure — your token is stored encrypted in the Android Keystore and is never exposed to web storage
• No cloud — a direct connection to your Home Assistant only
• Dark, light or system theme
• Available in English, Danish and Swedish
• Confirm-on-tap option to avoid accidental actions
• Resizable widgets that fit your launcher grid

GETTING STARTED
1. Open the app and enter your Home Assistant address
2. Paste a long-lived access token (the app shows you how to create one)
3. Add widgets from your launcher and pick the entities you want

REQUIREMENTS
• Android 8.0 or newer
• A reachable Home Assistant instance
• A long-lived access token

HA Widgets is an independent project and is not affiliated with the Home Assistant project or Nabu Casa.
```
